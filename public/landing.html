Platform Features
Everything you need to teach online
🔥
Firebase Firestore
Real-time database with automatic scaling. All user data, courses, progress, and quizzes stored in Firestore.
🎬
GitHub Video Storage
Upload videos to GitHub repo. No hosting fees. Videos stream via raw URLs. Supports MP4, WebM, MOV.
📊
Progress Tracking
Real-time lesson completion, per-course progress bars, XP points, and automatic certificate issuance.
📝
Timed Quizzes
Multi-question quizzes with time limits, pass marks, instant scored feedback and explanations.
🏆
Certificates
Auto-issued certificates on course completion. Each has a unique ID for verification.
⚡
XP & Streaks
Students earn XP for enrolling, completing lessons, passing quizzes. Daily streaks keep them motivated.
🔐
Firebase Auth
Secure login with Firebase Authentication. Role-based access — Student, Instructor, Admin.
🌐
Render Deployed
Hosted on Render.com — no domain needed. Student, admin, and API all served from one URL.
📢
Announcements
Post platform-wide announcements with Info, Urgent, and Event tags.
